# BR Architects Website

A modern, responsive architecture company website built with pure HTML and CSS. This project showcases a clean and professional design for an architecture firm's online presence.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern Navigation**: Clean navbar with smooth scrolling to sections
- **Project Portfolio**: Grid-based project showcase with hover effects
- **Team Profiles**: Professional team member display with contact options
- **Contact Form**: Functional contact form for client inquiries
- **Interactive Elements**: Hover effects and smooth transitions

## Sections

1. **Navigation Bar**
   - Brand logo with distinctive BR styling
   - Smooth scroll navigation links
   - Responsive menu design

2. **Hero Header**
   - Full-width architectural background image
   - Overlay brand presentation
   - Visually striking introduction

3. **Projects Section**
   - Grid layout of architectural projects
   - Image hover effects with project names
   - Organized portfolio display

4. **About Section**
   - Company description and philosophy
   - Team member profiles with:
     - Professional photos
     - Names and titles
     - Role descriptions
     - Contact buttons

5. **Contact Section**
   - Clean contact form with fields for:
     - Name
     - Email
     - Subject
     - Comment
   - Send message functionality

6. **Map Section**
   - Location visualization
   - Full-width map image

7. **Footer**
   - Credit to w3.css framework
   - Clean and minimal design

## Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Modern styling with flexbox/grid layouts
- **Responsive Design** - Mobile-first approach
- **Clean Architecture** - Well-organized code structure
